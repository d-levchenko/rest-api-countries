# REST Countries API Project

A React application that fetches country data from the REST Countries API and
displays it in a list.

## Features

- Fetches country data from the REST Countries API.
- Displays a list of countries with their name, population, region, and capital.
- Caches API responses to reduce the number of requests made.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/d-levchenko/rest-api-countries.git
```

2. Install dependencies:

```bash
cd rest-api-countries
npm install
```

3. Start the development server:

```bash
npm run dev
```

## Usage

The application will fetch country data from the REST Countries API and display
it in a list. Each country has a name, population, region, and capital.
